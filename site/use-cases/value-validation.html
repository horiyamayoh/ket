値の範囲と表記を小さいAPIに固定する

裸のintや文字列で扱うとぶれやすいdomain値を、moduleごとの値型やparseで固定する。

Decision Flow

  • port番号は port を使う。
  • IPv4 dotted decimalは ipv4 を使う。
  • version tripletは version を使う。
  • MAC addressとUUIDは mac、uuid を使う。
  • RGB hexは color を使う。
  • percent値は percent を使う。
  • 日付妥当性は date を使う。
  • 汎用overflowやcast境界は numeric を使う。

Recipe

  • 入力直後にdomain moduleへ変換する。
  • 変換後は裸値ではなく意味のある型や名前のAPIで扱う。
  • formatも同じmoduleへ戻して表記を固定する。
