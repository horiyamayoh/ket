Use This When

  • enumと文字列の対応表を明示してparse/formatしたい時。
  • enum class flagsのbit操作を読みやすくしたい時。

Do Not Use This When

  • reflectionや自動列挙名取得が必要な時。
  • 巨大なschema-driven enum変換を作りたい時。

HasFlag function

template <typename E> constexpr bool HasFlag(E flags, E flag) noexcept

指定flagがすべて立っているか判定。

Parameters

  • in flags: 判定対象のflag集合。
  • in flag: 判定するflag。

Return / retval

  • true: `flag`の全bitが`flags`に含まれる。
  • false: `flag`の少なくとも1bitが`flags`に含まれない。

Contract

prepost
`E`はunsigned underlying typeを持つbit mask enum型。引数と外部状態の変更なし。`flag`が0の場合はtrueを返す。

Notes

  • 0 maskは全bitが含まれる状態として扱う。

Example

enum class Permission : unsigned { kRead = 1U, kWrite = 2U, kReadWrite = 3U };
const auto has_write = ket::enums::HasFlag(Permission::kReadWrite, Permission::kWrite);
// has_write == true
