ket::enums

enum classのtable-based変換とflags操作を扱う。

Use This When

  • enumと文字列の対応表を明示してparse/formatしたい時。
  • enum class flagsのbit操作を読みやすくしたい時。

Do Not Use This When

  • reflectionや自動列挙名取得が必要な時。
  • 巨大なschema-driven enum変換を作りたい時。
