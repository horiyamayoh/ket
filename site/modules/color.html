ket::color

RGB値と6桁hex文字列を相互変換する。

Use This When

  • RGB 24bit値を小さい値型として扱いたい時。
  • 先頭#の有無やhex文字を固定したい時。

Do Not Use This When

  • alpha、色空間変換、CSS color全般を扱いたい時。
  • 画像処理や描画frameworkが必要な時。

TryParse function

inline bool TryParse(const char* text, std::size_t size, Rgb& out) noexcept

RGBの6桁hex文字列解析。

Parameters

  • in text: 解析対象の文字列。`RRGGBB`、または`#RRGGBB`を受け付ける。
  • in size: `text`のbyte数。終端NULは含めない。
  • out out: 成功時に解析後のRGB値を格納。

Return / retval

  • true: 6桁RGB hexとして解析成功。
  • false: null pointer、不正長、不正hex、3桁短縮形、またはalpha付き入力。

Contract

prepost
なし。null pointerや不正入力はfalse。成功時だけ`out`を更新。失敗時は`out`と外部状態の変更なし。

Notes

  • hex digitの大文字小文字混在は許容。CSS color nameとalphaは対象外。

Example

ket::color::Rgb color;
const auto parsed = ket::color::TryParse("#0aFF10", 7U, color);
// parsed == true, color.r == 0x0a, color.g == 0xff, color.b == 0x10

TryParse function

inline bool TryParse(const std::string& text, Rgb& out) noexcept

RGBの6桁hex文字列解析。

Parameters

  • in text: 解析対象の文字列。`RRGGBB`、または`#RRGGBB`を受け付ける。
  • out out: 成功時に解析後のRGB値を格納。

Return / retval

  • true: 6桁RGB hexとして解析成功。
  • false: 不正長、不正hex、3桁短縮形、またはalpha付き入力。

Contract

prepost
なし。任意のstd::stringを入力として扱い、不正入力はfalse。成功時だけ`out`を更新。失敗時は`out`と外部状態の変更なし。

Notes

  • hex digitの大文字小文字混在は許容。文字列リテラルを確保なしで扱う場合はpointer overloadを使用。

Example

ket::color::Rgb color;
const std::string text = "#0aFF10";
const auto parsed = ket::color::TryParse(text, color);
// parsed == true, color.r == 0x0a, color.g == 0xff, color.b == 0x10
