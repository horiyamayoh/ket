ket::byte_view

所有しないbyte範囲とslice境界を明示する。

Use This When

  • byte配列を所有せずに渡したい時。
  • nullptr+0とnullptr+非0の意味を固定したい時。

Do Not Use This When

  • payloadを所有して構築したい時。
  • C++20 std::span が利用でき、制約が十分に表現できる時。

API Details

View type

class View

読み取り専用のnon-owning byte view。

Notes

  • viewは元bufferを所有せず、利用者がbuffer lifetimeを保持。

size_ function

constexpr View(const std::uint8_t* data, std::size_t size) noexcept : data_(data), size_(size)

読み取り専用byte列を参照するview構築。

Parameters

  • in data: 参照対象bufferの先頭。`nullptr`は`size == 0`の場合のみ空view。
  • in size: 参照対象bufferのbyte数。

Contract

prepost
`data`は`size`byte以上読み取り可能な配列を指す。`nullptr + 非0`はinvalid viewとして保持。`data`と`size`を保持し、bufferの所有権は取得なし。

Example

const std::uint8_t data[] = {0x10U, 0x20U};
ket::byte_view::View view(data, 2U);
// view.Data() == data
// view.Size() == 2

TryAt function

bool TryAt(std::size_t index, std::uint8_t& out) const noexcept

指定indexのbyte取得。

Parameters

  • in index: 取得対象の0始まりindex。
  • out out: 取得したbyteの格納先。

Return / retval

  • true: viewがvalidかつ`index < Size()`で、`out`へbyteを格納。
  • false: invalid view、または範囲外index。`out`は変更なし。

Contract

prepost
`out`は有効な参照。viewの元buffer lifetimeは呼び出し中維持。成功時のみ`out`を変更。viewと元bufferの変更なし。

Example

const std::uint8_t data[] = {0x10U, 0x20U};
ket::byte_view::View view(data, 2U);
std::uint8_t value = 0U;
const bool ok = view.TryAt(1U, value);
// ok == true, value == 0x20

TrySlice function

bool TrySlice(std::size_t offset, std::size_t count, View& out) const noexcept

指定範囲の読み取り専用subview取得。

Parameters

  • in offset: subview開始位置。
  • in count: subview byte数。
  • out out: 取得したsubviewの格納先。

Return / retval

  • true: viewがvalidかつ`offset`から`count`byteが範囲内で、`out`へsubviewを格納。
  • false: invalid view、または範囲外slice。`out`は変更なし。

Contract

prepost
`out`は有効な参照。viewの元buffer lifetimeはsubview利用中も維持。成功時のみ`out`を変更。viewと元bufferの変更なし。

Notes

  • `offset == Size()`かつ`count == 0`の空subviewは成功。

Example

const std::uint8_t data[] = {0x10U, 0x20U, 0x30U};
ket::byte_view::View view(data, 3U);
ket::byte_view::View slice;
const auto ok = view.TrySlice(1U, 2U, slice);
// ok == true, slice.Data() == data + 1, slice.Size() == 2
