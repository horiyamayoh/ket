optionalの値変換、合成、遅延fallback評価を扱う。

Use This When

  • optional保持値をmap/and_then風に合成したい時。
  • fallbackを必要時だけ評価したい時。

Do Not Use This When

  • C++23標準optional memberで十分な時。
  • error理由を保持したい時。

API Details

Map function

Signature
template <typename T, typename F> constexpr auto Map(std::optional<T>& value, F&& f)
Briefmutable optional保持値の変換。
Parameters
  • in,out value: 変換対象のoptional。値を持つ場合はmapperへmutable参照を渡す。
  • in f: 保持値を変換する`std::invoke`互換mapper。
Return / retval
  • value: `value`が値を持つ場合、`std::invoke(f, *value)`を`std::decay_t`で保持するoptional。
  • std::nullopt: `value`が空の場合。
Pre`std::invoke(f, *value)`はvoidではないobjectを値として返す。直接の参照戻り値は不可。
Post空の場合は`f`を呼ばない。値ありの場合、mapperは`T&`を通じて保持値を変更できる。
Notes
  • 参照を保持したい場合は、mapperが`std::reference_wrapper<T>`を明示的に返す。
  • mapperの副作用と例外は、呼び出し元が渡したcallableの規約に従う。
  • C++23以降では`std::optional::transform`が標準代替。
Example
auto source = std::optional<int>(2);
const auto value = ket::optional::Map(source, [](int& x) {
    x += 1;
    return x * 3;
});
// source == std::optional<int>(3), value == std::optional<int>(9)

  • C++23以降では`std::optional::and_then`が標準代替。
Example
auto source = std::optional<int>(2);
const auto value = ket::optional::AndThen(source, [](int& x) {
    x += 1;
    return std::optional<int>(x * 3);
});
// source == std::optional<int>(3), value == std::optional<int>(9)

ValueOrEval function

Signature
template <typename T, typename F> constexpr T ValueOrEval(const std::optional<T>& value, F&& fallback_factory)
Briefoptional保持値、または遅延評価fallbackの値取得。
Parameters
  • in value: 値取得対象のoptional。
  • in fallback_factory: 空の場合だけ評価する`std::invoke`互換fallback factory。
Return / retval
  • value: `value`が値を持つ場合はその保持値のcopy、空の場合は`fallback_factory()`の結果。
PreTはcopy constructible。`std::invoke(fallback_factory)`はTへ変換可能な値を返す。
Post値ありの場合は`fallback_factory`を呼ばない。API自身は`value`の保持状態を変更しない。
Notes
  • `std::optional::value_or`と異なり、fallbackは必要時だけ評価。
  • fallback factoryの副作用と例外は、呼び出し元が渡したcallableの規約に従う。
Example
const auto value = ket::optional::ValueOrEval(std::optional<int>(), [] { return 42; });
// value == 42
