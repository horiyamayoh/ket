API Details

Cat function

template <typename... Parts> std::string Cat(const Parts&... parts)

複数文字列片の連結。

Parameters

  • in parts: 連結対象の文字列片。

Return / retval

  • value: 連結後の文字列。

Contract

prepost
各要素はstd::string_viewに変換可能な文字列片、またはchar。raw C stringは非nullかつnull終端。引数と外部状態の変更なし。出力文字列は入力順と入力内容を保持。

Notes

  • 数値、enum、任意stream変換は対象外。必要な変換は呼び出し側で明示。
  • 合計長がstd::string::max_size()を超える場合はstd::length_error送出。
  • std::stringのreserveとappendがallocationを伴うためnoexceptなし。

Example

const auto text = ket::string::Cat("id=", id_text, ", mode=", mode_text);
// text == "id=42, mode=run"

Append function

template <typename... Parts> void Append(std::string& destination, const Parts&... parts)

複数文字列片の末尾追記。

Parameters

  • in,out destination: 追記先の文字列。
  • in parts: 追記対象の文字列片。

Return / retval

  • void: 戻り値なし。

Contract

prepost
destinationは有効なstd::string。各要素はstd::string_viewへ変換可能、またはchar。 raw C stringは非nullかつnull終端。`destination`の既存内容を保持し、末尾に`parts`の連結結果を追加。

Notes

  • `destination`自身やそのviewをpartsに含む場合は、suffixを生成してから追記。
  • 合計長がstd::string::max_size()を超える場合はstd::length_error送出。
  • std::stringのreserve、生成、appendがallocationを伴うためnoexceptなし。

Example

std::string text = "id=";
ket::string::Append(text, id_text, ", mode=", mode_text);
// text == "id=42, mode=run"
