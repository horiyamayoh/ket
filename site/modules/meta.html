ket::meta

C++11/14で欠けるtype traitsを小さく補う。

Use This When

  • remove_cvrefやvoid_tなどの小さい型操作をC++11/14で使いたい時。
  • templateコードの意図を短い名前へ寄せたい時。

Do Not Use This When

  • conceptsや大規模なdetection frameworkが必要な時。
  • 標準type traitsで十分に明確な時。

API Details

TypeIdentity type

template <typename T> struct TypeIdentity

`T`を`type`としてそのまま返すidentity type trait。

Notes

  • C++20 `std::type_identity` の小さいC++11/14向け補助。

Example

using Value = ket::meta::TypeIdentity<const int&>::type;
// Value == const int&

AlwaysFalse type

template <typename... Types> struct AlwaysFalse : std::false_type

任意のtemplate引数に対して常にfalseとなるtype trait。

Notes

  • dependent falseが必要なstatic_assertでtemplate diagnosticsを保つための補助。

Example

static_assert(!ket::meta::AlwaysFalse<int>::value, "not instantiated failure");

VoidT alias

template <typename... Types> using VoidT = typename detail::MakeVoid<Types...>::type

任意のtemplate引数列を`void`へ写像するalias。

Notes

  • C++17 `std::void_t` のC++11/14向け補助。

Example

using Value = ket::meta::VoidT<int, double>;
// Value == void
