ket::build_config

compiler、OS、標準ライブラリfeatureの差分をKET_* macroへ集約する。

Module Facts

Headermodules/build_config/ket_build_config.h
Statusverified
C++ MinC++11
TestsGoogleTest + C++11/14/20/23 compile-only
Progress notecompiler/OS/標準feature macroを0/1で判定
Drop-in`ket_build_config.h` を対象プロジェクトへコピー。ヘッダオンリーmodule。
Dependencies標準ライブラリ、compiler predefined macro、Apple platform では `TargetConditionals.h`。 他のket moduleへの依存なし。
Namespace公開API：global KET_* macros（C++ namespaceなし）。 内部実装：KET_DETAIL_* macros（ヘッダ末尾で#undef）。
CategoriesC++言語・型補助
Use casestemplate、tuple、variantの儀式を短くする

Use This When

  • C++標準、compiler、OS、feature-test macroの差分を小さく閉じたい時。
  • drop-in先でpreprocessor分岐の意図を読みやすくしたい時。

Do Not Use This When

  • build systemやpackage manager全体の設定を扱いたい時。
  • runtimeのplatform検出や詳細なCPU feature検出が必要な時。

  • runtime cost、allocation、link symbolは発生しない。
