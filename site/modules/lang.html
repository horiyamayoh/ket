API Details

IgnoreUnused function

template <typename... Args> void IgnoreUnused(const Args&... args) noexcept

未使用値の明示破棄。

Parameters

  • in args: 未使用として扱う値。

Return / retval

  • void: 戻り値なし。

Contract

prepost
引数式は呼び出し前に評価済み。副作用の有無は呼び出し側の責任。引数と外部状態の変更なし。参照束縛のみでコピーやムーブなし。

Notes

  • C++17 `[[maybe_unused]]` と一部重複するが、式の明示破棄用途として採用。

Example

ket::lang::IgnoreUnused(argc, argv);

T function

template <typename T, std::size_t N> constexpr std::size_t ArraySize(const T (&array)[N]) noexcept

raw配列の要素数取得。

Parameters

  • in array: 要素数を取得するraw配列。

Return / retval

  • value: `array`の要素数。

Contract

prepost
`array`は有効なraw配列参照。ポインタ、動的配列、空配列は対象外。引数と外部状態の変更なし。

Notes

  • C++17 `std::size` の限定的なC++11代替。

Example

int values[3] = {};
const auto count = ket::lang::ArraySize(values);
// count == 3

AsConst function

template <typename T> constexpr typename std::add_const<T>::type& AsConst(T& value) noexcept

lvalueのconst参照取得。

Parameters

  • in value: const参照として扱うlvalue。

Return / retval

  • value: `value`と同じobjectを参照するconst参照。

Contract

prepost
`value`は呼び出し後も参照が有効なlvalue。rvalueは受け付けない。引数と外部状態の変更なし。戻り値は入力objectの寿命に従う。

Notes

  • C++17 `std::as_const` のC++11代替。

Example

int value = 42;
const int& view = ket::lang::AsConst(value);
