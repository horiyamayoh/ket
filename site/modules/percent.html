ket::percent

percent、ratio、basis pointsを0〜100%の値型へ正規化する。

Namespace公開API：ket::percent 内部実装：ket::percent::detail

Use This When

  • 0〜100%の範囲と丸めを呼び出し側ごとにぶらしたくない時。
  • basis points単位でpercent値を保持したい時。

Do Not Use This When

  • 100%を超える割合や負の割合を扱う時。
  • unit frameworkや任意精度decimalが必要な時。

API Details

Percent type

class Percent

0〜100%を1 basis point単位で保持する値型。

Notes

  • 保持範囲は0から10000 basis points。1 basis pointは0.01%。

TryFromBasisPoints function

static bool TryFromBasisPoints(std::uint32_t basis_points, Percent& out) noexcept

basis points単位からPercent値を構築。

Parameters

  • in basis_points: 入力basis points。0から10000のみ成功。
  • out out: 成功時に構築結果を書き込む出力先。

Return / retval

  • true: `basis_points <= 10000`。
  • false: `basis_points > 10000`。`out`は変更なし。

Contract

prepost
`out`は有効なPercentオブジェクト。成功時のみ`out`を`basis_points`相当へ更新。失敗時は`out`を保持。

Example

ket::percent::Percent value;
const auto ok = ket::percent::Percent::TryFromBasisPoints(1234U, value);
// ok == true, value.ToPercent() == 12.34

TryFromPercent function

static bool TryFromPercent(double percent, Percent& out) noexcept

percent単位の小数からPercent値を構築。

Parameters

  • in percent: 入力percent値。0.0から100.0のみ成功。
  • out out: 成功時に構築結果を書き込む出力先。

Return / retval

  • true: 有限かつ0.0から100.0の入力。
  • false: 範囲外、NaN、Inf。`out`は変更なし。

Contract

prepost
`out`は有効なPercentオブジェクト。成功時のみ`out`をnearest basis pointへ丸めた値に更新。失敗時は`out`を保持。

Notes

  • 丸めは入力`double`値をbasis pointsへscaleした後の`floor(value + 0.5)`相当。 十進表記の厳密なtie丸めが必要な場合は`TryFromBasisPoints`を使用。

Example

ket::percent::Percent value;
const auto ok = ket::percent::Percent::TryFromPercent(12.345, value);
// ok == true, value.BasisPoints() == 1235

FromPercentClamped function

static Percent FromPercentClamped(double percent) noexcept

percent単位の小数を範囲内へ丸め込んでPercent値を構築。

Parameters

  • in percent: 入力percent値。

Return / retval

  • value: 0.0から100.0へclamp後、nearest basis pointへ丸めたPercent値。

Contract

prepost
なし。NaNは0%、-Infは0%、+Infは100%として扱う。引数とketオブジェクトの変更なし。

Notes

  • 有限の範囲外入力は0.0から100.0へclamp。丸めは入力`double`値をbasis pointsへscaleした後の`floor(value + 0.5)`相当。

Example

const auto value = ket::percent::Percent::FromPercentClamped(120.0);
// value.BasisPoints() == 10000
