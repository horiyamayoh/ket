ket::platform

errno、Windows error、environment variableを標準文字列で扱う。

Use This When

  • platform error codeを人間向けmessageへ変換したい時。
  • process environment variableをOS差分込みで小さく読みたい時。

Do Not Use This When

  • process管理、terminal制御、OS抽象化layerが必要な時。
  • 環境変数の永続変更や設定frameworkを作りたい時。

API Details

FormatErrno function

Signature
std::string FormatErrno(int error_number)
Brieferrno番号のplatform message文字列化。
Parameters
  • in error_number: 文字列化対象のerrno番号。
Return / retval
  • value: platformのthread-safe APIから取得したmessage、またはASCII fallback。
Preなし。未知のerrno番号はfallback文字列として扱う。
Post引数と外部状態の変更なし。戻り値は空文字列にしない。
Notes
  • message取得に失敗した場合は`Unknown error <n>`形式を返す。
Example
const auto text = ket::platform::FormatErrno(22);
// !text.empty()

ReadEnvironmentVariable function

Signature
std::optional<std::string> ReadEnvironmentVariable(std::string_view name)
Briefprocess environment variableの取得。
Parameters
  • in name: 取得対象のenvironment variable名。
Return / retval
  • value: environment variableが存在する場合の値。
  • std::nullopt: missing、空name、NULを含むname、またはplatform変換失敗。
Preなし。空nameとNULを含むnameは失敗値として扱う。
Postprocess environmentは変更しない。Windowsでは現在threadのlast-error codeを保持する。 environment同時変更時の一貫性はplatform API規約に従う。
Notes
  • Windowsではwide APIからUTF-8 narrow stringへ変換する。
Example
const auto value = ket::platform::ReadEnvironmentVariable("");
// value == std::nullopt

FormatWindowsError function

Signature
std::string FormatWindowsError(WindowsErrorCode code)
BriefWindows error codeのmessage文字列化。
Parameters
  • in code: 文字列化対象のWindows error code。
Return / retval
  • value: Windows wide APIから取得し、末尾の空白・改行を除去してUTF-8へ変換したmessage、 またはASCII fallback。
Preなし。未知のWindows error codeはfallback文字列として扱う。
Post引数と外部状態の変更なし。戻り値は空文字列にしない。
Notes
  • UTF-8変換に失敗した場合は`Unknown Windows error <n>`形式を返す。
  • Windows message resource由来の末尾CR、LF、tab、space、NULは除去する。
Example
const auto text = ket::platform::FormatWindowsError(
    static_cast<ket::platform::WindowsErrorCode>(2));
// !text.empty()
