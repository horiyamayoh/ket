ket::ascii

ASCII前提のtrim、split、case変換、置換を小さいAPIへ分離する。

Module Facts

Statusverified
C++ MinC++17

Progress noteASCII前提のtrim、split、case変換、置換
Drop-in`ket_ascii.h` と `ket_ascii.cpp` を対象プロジェクトへコピー。
Dependencies標準ライブラリのみ。 他のket moduleへの依存なし。
Namespace公開API：ket::ascii 内部実装：ket::ascii::detail
Categories文字列・text・parse 入力境界・外部境界
Use cases文字列の小処理を読みやすくする CLI引数や設定文字列を読む

Use This When

  • 入力がASCII前提であることを呼び出し側で固定できる時。
  • trimやsplitの手順より、ASCII限定の意図を読みたい時。

Do Not Use This When

  • Unicodeのcase foldingやgrapheme clusterを扱う時。
  • locale依存の文字分類が必要な時。
