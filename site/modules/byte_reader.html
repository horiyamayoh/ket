ket::byte_reader

non-owning bufferから成功時だけoffsetを進めてbyte列を読む。

Use This When

  • 固定bufferから順にfieldを読みたい時。
  • size不足時にoffsetと出力を不変にしたい時。

Do Not Use This When

  • stream I/Oや非同期I/Oを扱う時。
  • buffer所有や再確保が必要な時。

Views

バイナリ・byte列固定bufferから安全に読む・書く 小さい電文payloadを組み立てる

API Details

Reader type

class Reader

固定bufferを所有せずに読み進めるreader。

Notes

  • `data == nullptr && size == 0`は有効な空reader。
  • `data == nullptr && size > 0`はinvalid readerとして読み取り失敗。

Reader function

Reader(const std::uint8_t* data, std::size_t size) noexcept

non-owning buffer readerの構築。

Parameters

  • in data: 読み取り対象buffer先頭。`size == 0`の場合のみnullptr可。
  • in size: `data`のbyte数。

Contract

prepost
`data != nullptr`の場合、`data`は`size`byte以上読み取り可能な配列を指す。 `data == nullptr && size > 0`はprecondition違反ではなくinvalid readerとして保持。`Offset()`は0。入力pointerとsizeを保持。

Example

const std::uint8_t data[] = {0x12U, 0x34U};
ket::byte_reader::Reader reader(data, 2U);
// reader.Size() == 2
// reader.Offset() == 0

Reader function

Reader(const Reader& other) noexcept = default

reader状態のcopy構築。

Parameters

  • in other: copy元reader。

Contract

prepost
なし。non-owning pointer、size、offsetを`other`と同じ値で保持。

Example

const std::uint8_t data[] = {0x12U};
ket::byte_reader::Reader source(data, 1U);
ket::byte_reader::Reader copied(source);
// copied.Size() == 1
// copied.Offset() == 0

Skip function

bool Skip(std::size_t size) noexcept

指定byte数の読み飛ばし。

Parameters

  • in size: 読み飛ばすbyte数。

Return / retval

  • true: `size`byteの読み飛ばし成功。
  • false: invalid reader、または残りbyte不足。

Contract

prepost
元buffer lifetimeはreader利用中保持。成功時だけ`Offset()`が`size`増加。失敗時はoffset不変。

Example

const std::uint8_t data[] = {0x12U, 0x34U};
ket::byte_reader::Reader reader(data, 2U);
const auto ok = reader.Skip(1U);
// ok == true
// reader.Offset() == 1

ReadU8 function

bool ReadU8(std::uint8_t& out) noexcept

1byte unsigned値の読み取り。

Parameters

  • out out: 読み取った値。

Return / retval

  • true: 1byte読み取り成功。
  • false: invalid reader、または残りbyte不足。

Contract

prepost
`out`は有効な参照。元buffer lifetimeはreader利用中保持。成功時だけ`Offset()`が1増加し、`out`へ値を書き込む。失敗時はoffsetと`out`不変。

Example

const std::uint8_t data[] = {0x12U};
ket::byte_reader::Reader reader(data, 1U);
std::uint8_t value = 0U;
const auto ok = reader.ReadU8(value);
// ok == true, value == 0x12U, reader.Offset() == 1

ReadBytes function

bool ReadBytes(std::size_t size, const std::uint8_t*& out_data) noexcept

指定byte数のnon-owning範囲読み取り。

Parameters

  • in size: 取り出すbyte数。
  • out out_data: 読み取り開始位置を指すnon-owning pointer。

Return / retval

  • true: `size`byteの範囲読み取り成功。
  • false: invalid reader、または残りbyte不足。

Contract

prepost
`out_data`は有効な参照。返されたpointerのlifetimeは元bufferに従う。成功時だけ`Offset()`が`size`増加し、`out_data`へ元buffer内の位置を書き込む。 失敗時はoffsetと`out_data`不変。

Notes

  • validな空readerから0byteを読む場合、`out_data`はnullptr。

Example

const std::uint8_t data[] = {0x10U, 0x20U, 0x30U};
ket::byte_reader::Reader reader(data, 3U);
const std::uint8_t* bytes = nullptr;
const auto ok = reader.ReadBytes(2U, bytes);
// ok == true, bytes == data, reader.Offset() == 2
