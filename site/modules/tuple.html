API Details

ForEach function

template <typename Tuple, typename F> void ForEach(Tuple&& tuple, F&& f)

tuple要素へのindex順の副作用呼び出し。

Parameters

  • in,out tuple: 走査対象のtuple-like object。non-const要素やrvalue要素は`f`により変更またはmoveされ得る。
  • in,out f: 各要素を1引数で受け取るcallable。渡されたcallable objectを直接呼び出し、API内でcopyしない。

Return / retval

  • void: 戻り値なし。

Contract

prepost
`tuple`はstd::applyで展開可能。`f`はlvalue callable objectとして各転送済み要素型で呼び出し可能。`f`はtuple要素のindex昇順に最大1回ずつ同じobjectとして呼び出される。callable例外は呼び出し元へ伝播。

Notes

  • `f`の戻り値は破棄。tuple要素のvalue categoryとcv修飾はstd::applyの規則に従う。

Example

auto values = std::make_tuple(1, 2, 3);
int sum = 0;
ket::tuple::ForEach(values, [&sum](int value) { sum += value; });
// sum == 6

Transform function

template <typename Tuple, typename F> auto Transform(Tuple&& tuple, F&& f)

tuple要素のindex順変換。

Parameters

  • in,out tuple: 変換対象のtuple-like object。non-const要素やrvalue要素は`f`により変更またはmoveされ得る。
  • in,out f: 各要素を1引数で受け取り、変換後の要素を返すcallable。渡されたcallable objectを直接呼び出し、API内でcopyしない。

Return / retval

  • value: 各要素への`f`の戻り値を同じindex順に格納したstd::tuple。

Contract

prepost
`tuple`はstd::applyで展開可能。`f`はlvalue callable objectとして各転送済み要素型で呼び出し可能で、戻り値はstd::tuple要素として構成可能。戻りtupleの参照要素を使う場合、参照先は戻りtupleの利用中に有効。`f`はtuple要素のindex昇順に最大1回ずつ同じobjectとして呼び出される。引数と外部状態は`f`の副作用以外で変更なし。

Notes

  • callable例外は呼び出し元へ伝播。`f`が参照を返す場合、戻り値は参照要素を持つtuple。rvalue tuple要素への参照を返すcallableではdangling referenceに注意。

Example

const auto values = std::make_tuple(1, 2, 3);
const auto doubled = ket::tuple::Transform(values, [](int value) { return value * 2; });
// std::get<0>(doubled) == 2, std::get<2>(doubled) == 6
