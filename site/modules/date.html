ket::date

日付、時刻、月日数、leap yearの境界条件を扱う。

Use This When

  • C++11/14/17で小さい日付妥当性だけが必要な時。
  • 1900/2000年や2月末の境界を手書きしたくない時。

Do Not Use This When

  • C++20 chrono calendarで明確に書ける時。
  • time zoneやlocale calendarを扱う時。

API Details

IsLeapYear function

constexpr bool IsLeapYear(int year) noexcept

Gregorian calendarのleap year判定。

Parameters

  • in year: 判定対象の年。

Return / retval

  • true: `year >= 1`、かつ4で割り切れ、100で割り切れないか400で割り切れる年。
  • false: `year < 1`、またはGregorian calendar上の平年。

Contract

prepost
なし。`year < 1`は無効な年としてfalse。引数と外部状態の変更なし。

Notes

  • timezoneとleap secondは対象外。

Example

const auto leap = ket::date::IsLeapYear(2000);
// leap == true

IsValidMonth function

constexpr bool IsValidMonth(unsigned month) noexcept

Gregorian calendarの月番号妥当性判定。

Parameters

  • in month: 判定対象の月番号。

Return / retval

  • true: `1 <= month <= 12`。
  • false: 0、または13以上。

Contract

なし。月番号は1始まりとして判定。引数と外部状態の変更なし。

Example

const auto valid = ket::date::IsValidMonth(12U);
// valid == true

TryGetDaysInMonth function

inline bool TryGetDaysInMonth(int year, unsigned month, unsigned& out) noexcept

Gregorian calendarの指定年月に含まれる日数取得。

Parameters

  • in year: 日数を取得する年。
  • in month: 日数を取得する月番号。
  • out out: 成功時に当月日数を書き込む出力引数。

Return / retval

  • true: `year >= 1`かつ`1 <= month <= 12`で、`out`へ日数を書き込んだ。
  • false: 無効な年または月。`out`は変更なし。

Contract

prepost
`out`は書き込み可能なunsignedオブジェクトへの参照。成功時のみ`out`を28、29、30、31のいずれかへ変更。失敗時は`out`を変更なし。

Notes

  • C++11では出力引数を変更するTry APIをconstexpr化できないためnoexceptのみ。

Example

unsigned days = 0U;
const auto ok = ket::date::TryGetDaysInMonth(2024, 2U, days);
// ok == true, days == 29

IsValidDate function

constexpr bool IsValidDate(int year, unsigned month, unsigned day) noexcept

Gregorian calendarの年月日妥当性判定。

Parameters

  • in year: 判定対象の年。
  • in month: 判定対象の月番号。
  • in day: 判定対象の日。

Return / retval

  • true: `year >= 1`、有効な月、かつ当月日数内の日。
  • false: 無効な年、月、0日、または月末を超える日。

Contract

prepost
なし。timezone、calendar conversion、date arithmeticは対象外。引数と外部状態の変更なし。

Notes

  • constexprを保つため、内部constexpr helperで当月日数を判定。

Example

const auto valid = ket::date::IsValidDate(2024, 2U, 29U);
// valid == true

IsValidTime function

constexpr bool IsValidTime(unsigned hour, unsigned minute, unsigned second) noexcept

24時間表記の時分秒妥当性判定。

Parameters

  • in hour: 判定対象の時。
  • in minute: 判定対象の分。
  • in second: 判定対象の秒。

Return / retval

  • true: `hour < 24`、`minute < 60`、`second < 60`。
  • false: 時、分、秒のいずれかが範囲外。

Contract

prepost
なし。leap secondは扱わず、`second == 60`は無効。引数と外部状態の変更なし。

Example

const auto valid = ket::date::IsValidTime(23U, 59U, 59U);
// valid == true

IsValidDateTime function

constexpr bool IsValidDateTime(int year, unsigned month, unsigned day, unsigned hour, unsigned minute, unsigned second) noexcept

Gregorian calendarの日付と24時間表記の時分秒を合わせた妥当性判定。

Parameters

  • in year: 判定対象の年。
  • in month: 判定対象の月番号。
  • in day: 判定対象の日。
  • in hour: 判定対象の時。
  • in minute: 判定対象の分。
  • in second: 判定対象の秒。

Return / retval

  • true: 年月日と時分秒がどちらも有効。
  • false: 年月日、または時分秒のいずれかが範囲外。

Contract

prepost
なし。timezone、calendar conversion、date arithmetic、leap secondは対象外。引数と外部状態の変更なし。

Notes

  • `IsValidDate`と`IsValidTime`を組み合わせた判定。

Example

const auto valid = ket::date::IsValidDateTime(2024, 2U, 29U, 23U, 59U, 59U);
// valid == true
