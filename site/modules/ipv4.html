ket::ipv4

IPv4 dotted decimalとbig-endian 32bit表現を相互変換する。

Use This When

  • IPv4 literalを範囲確認して値として扱いたい時。
  • BE 32bit表現との変換を明示したい時。

Do Not Use This When

  • IPv6、CIDR、DNS解決、socket addressを扱う時。
  • OS network API wrapperが必要な時。

API Details

Address type

struct Address

IPv4 addressの4 octet値。

Notes

  • `octets[0]`がdotted decimalの先頭要素、またはBE 32bit表現の最上位octet。 既定値は`0.0.0.0`。

Parse function

std::optional<Address> Parse(std::string_view text) noexcept

dotted decimal文字列のIPv4 address変換。

Parameters

  • in text: 変換対象のdotted decimal文字列。

Return / retval

  • value: 変換後のIPv4 address。
  • std::nullopt: octet範囲外、負値、空要素、要素不足/過多、空白、またはleading zero。

Contract

prepost
なし。不正なdotted decimalは失敗値として扱う。引数と外部状態の変更なし。

Notes

  • dotted decimalのみを受け付け、IPv6、CIDR、DNS名、port付き表記は扱わない。

Example

const auto value = ket::ipv4::Parse("192.168.0.1");
// value->octets == {192, 168, 0, 1}

Format function

std::string Format(Address value)

IPv4 addressのdotted decimal文字列変換。

Parameters

  • in value: 変換対象のIPv4 address。

Return / retval

  • value: dotted decimal文字列。

Contract

prepost
なし。任意の4 octet値をIPv4 addressとして扱う。引数と外部状態の変更なし。

Notes

  • std::stringの確保があるためnoexceptなし。

Example

const ket::ipv4::Address address = {{192, 168, 0, 1}};
const auto text = ket::ipv4::Format(address);
// text == "192.168.0.1"

ToBe32 function

constexpr std::uint32_t ToBe32(Address value) noexcept

IPv4 addressのBE 32bit整数変換。

Parameters

  • in value: 変換対象のIPv4 address。

Return / retval

  • value: `octets[0]`を上位8bitへ置く32bit整数表現。

Contract

prepost
なし。`octets[0]`を最上位octetとして扱う。引数と外部状態の変更なし。

Notes

  • host/network byte order変換ではなく、IPv4 addressをbig-endian順の数値へ詰める処理。

Example

const ket::ipv4::Address address = {{192, 168, 0, 1}};
const auto value = ket::ipv4::ToBe32(address);
// value == 0xC0A80001

FromBe32 function

constexpr Address FromBe32(std::uint32_t value) noexcept

BE 32bit整数のIPv4 address変換。

Parameters

  • in value: `octets[0]`を上位8bitに置いた32bit整数表現。

Return / retval

  • value: 変換後のIPv4 address。

Contract

prepost
なし。最上位byteを`octets[0]`へ展開する。引数と外部状態の変更なし。

Notes

  • host/network byte order変換ではなく、big-endian順の32bit数値表現から4 octetへ展開する処理。

Example

const auto address = ket::ipv4::FromBe32(0xC0A80001U);
// address.octets == {192, 168, 0, 1}
