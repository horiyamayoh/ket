ket::ipv4

IPv4 dotted decimalとbig-endian 32bit表現を相互変換する。

Use This When

  • IPv4 literalを範囲確認して値として扱いたい時。
  • BE 32bit表現との変換を明示したい時。

Do Not Use This When

  • IPv6、CIDR、DNS解決、socket addressを扱う時。
  • OS network API wrapperが必要な時。

Parse function

Signature
std::optional<Address> Parse(std::string_view text) noexcept
Briefdotted decimal文字列のIPv4 address変換。
Parameters
  • in text: 変換対象のdotted decimal文字列。
Return / retval
  • value: 変換後のIPv4 address。
  • std::nullopt: octet範囲外、負値、空要素、要素不足/過多、空白、またはleading zero。
Preなし。不正なdotted decimalは失敗値として扱う。
Post引数と外部状態の変更なし。
Notes
  • dotted decimalのみを受け付け、IPv6、CIDR、DNS名、port付き表記は扱わない。
Example
const auto value = ket::ipv4::Parse("192.168.0.1");
// value->octets == {192, 168, 0, 1}

ToBe32 function

Signature
constexpr std::uint32_t ToBe32(Address value) noexcept
BriefIPv4 addressのBE 32bit整数変換。
Parameters
  • in value: 変換対象のIPv4 address。
Return / retval
  • value: `octets[0]`を上位8bitへ置く32bit整数表現。
Preなし。`octets[0]`を最上位octetとして扱う。
Post引数と外部状態の変更なし。
Notes
  • host/network byte order変換ではなく、IPv4 addressをbig-endian順の数値へ詰める処理。
Example
const ket::ipv4::Address address = {{192, 168, 0, 1}};
const auto value = ket::ipv4::ToBe32(address);
// value == 0xC0A80001
