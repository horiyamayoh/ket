steady_clockベースのStopwatchとDeadlineを扱う。

Use This When

  • timeoutや経過時間をsystem_clockと混ぜずに扱いたい時。
  • 期限切れ判定を小さい値型に閉じたい時。

Do Not Use This When

  • wall-clock時刻やcalendar日付を扱う時。
  • async schedulerやevent loopが必要な時。

API Details

Stopwatch type

class Stopwatch

steady_clockで経過時間を測る小さいstopwatch。

開始時刻だけを保持するvalue object。待機、timer thread、同期制御は行わない。同一objectを複数threadからRestartしながら読む場合は呼び出し側で同期。

Example

auto stopwatch = ket::deadline::Stopwatch::StartNew();
const auto elapsed = stopwatch.Elapsed();

Restart function

void Restart() noexcept

開始点を現在時刻へ戻す。

Return / retval

  • void: 戻り値なし。

Contract

prepost
なし。以後の`Elapsed()`はrestart後の経過時間を返す。

stopwatch.Restart();
const auto elapsed = stopwatch.Elapsed();

Deadline type

class Deadline

steady_clockで表す期限時刻。

`std::chrono::steady_clock::time_point`だけを保持するvalue object。待機、timer thread、同期制御は行わない。const member functionはobjectを変更しない。同一objectを複数threadで代入・参照する場合は呼び出し側で同期。

Example

const auto deadline = ket::deadline::Deadline::After(std::chrono::seconds(5));
const auto expired = deadline.Expired();

After function

static Deadline After(std::chrono::steady_clock::duration timeout) noexcept

現在時刻からtimeout後のdeadline生成。

Parameters

  • in timeout: 現在時刻へ加算する`steady_clock` duration。

Return / retval

  • value: timeout後のdeadline。zero以下のtimeoutは即時期限切れ。 正timeoutが`std::chrono::steady_clock::time_point::max()`を超える場合は `std::chrono::steady_clock::time_point::max()`へ飽和。

Contract

prepost
なし。負timeoutは即時期限切れとして扱う。引数と外部状態の変更なし。正timeoutでは生成時に取得した現在時刻以後の期限を保持。 zero以下のtimeoutでは生成直後から期限切れ。

Example

const auto deadline = ket::deadline::Deadline::After(std::chrono::seconds(60));
const auto expired = deadline.Expired();
// expired == false

At function

static Deadline At(std::chrono::steady_clock::time_point time_point) noexcept

指定したsteady_clock時刻のdeadline生成。

Parameters

  • in time_point: 期限として保持する`steady_clock`時刻。

Return / retval

  • value: `time_point`を期限にしたdeadline。

Contract

prepost
`time_point`は`std::chrono::steady_clock`由来の時刻。引数と外部状態の変更なし。

Example

const auto deadline = ket::deadline::Deadline::At(time_point);
