ket::bcd

packed BCDと10進整数・10進文字列の相互変換を扱う。

Use This When

  • 電文や古い外部仕様でpacked BCD fieldが必要な時。
  • nibble妥当性や先頭ゼロ保持を呼び出し側に散らしたくない時。

Do Not Use This When

  • 通常の10進文字列formatだけで十分な時。
  • 任意精度decimal演算や金額計算をしたい時。

Notes

  • 整数変換では先頭ゼロを保持しない。桁保持が必要なら文字列変換を使う。

Post引数と外部状態の変更なし。
Notes
  • 入力BCDの桁数を保ち、先頭の0も文字'0'として出力。
  • std::stringの確保があるためnoexceptなし。
Example
const std::uint8_t data[] = {0x00U, 0x42U};
const auto value = ket::bcd::Format(data, 2U);
// value == std::optional<std::string>("0042")
