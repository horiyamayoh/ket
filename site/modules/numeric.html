ket::numeric

整数overflow、alignment、safe castの境界確認を扱う。

Module Facts

Headermodules/numeric/ket_numeric.h
Statusverified
C++ MinC++11
TestsGoogleTest + C++11 compile-only
Progress noteoverflow、align、cast の境界確認
Drop-in`ket_numeric.h` を対象プロジェクトへコピー。ヘッダオンリーmodule。
Dependencies標準ライブラリのみ。 他のket moduleへの依存なし。
Namespace公開API：ket::numeric 内部実装：ket::numeric::detail
Categoriesバイナリ・byte列 値型・validation
Use cases小さい電文payloadを組み立てる 値の範囲と表記を小さいAPIに固定する

Use This When

  • signed/unsignedや幅変換の境界を毎回書きたくない時。
  • overflowやalignment 0を失敗として扱いたい時。

Do Not Use This When

  • 任意精度整数やdecimal演算が必要な時。
  • 標準の単純なcastで意図が十分明確な時。

Notes

const auto ng = ket::numeric::InRange<unsigned>(-1);
// ok == true, ng == false
