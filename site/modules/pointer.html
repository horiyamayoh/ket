ket::pointer

所有しないraw pointerのnull不許可、weak pointer lock、address取得を扱う。

Use This When

  • raw pointerがnullでない前提を型名で明示したい時。
  • weak pointerのlock結果やoverloadされたoperator&回避を短く扱いたい時。

Do Not Use This When

  • 所有権を表すsmart pointer wrapperが必要な時。
  • lifetimeやborrow checkingをframeworkとして管理したい時。

API Details

NotNull type

Signature
template <typename T> class NotNull
Briefnullを保持しないraw pointer wrapper。
Notes
  • 所有権は持たず、参照先のlifetimeは呼び出し側の責任。
  • `void`は対象外。class instantiation時のstatic_assertで拒否。 参照先型があるraw pointerに限定。

NotNull function

Signature
constexpr explicit NotNull(T* ptr)
Briefnon-null raw pointerからの構築。
Parameters
  • in ptr: 保持対象のraw pointer。
Pre`ptr`はraw pointer。`nullptr`はstd::invalid_argumentとして扱う。
Post構築成功時、`Get()`は`ptr`と同じpointer値を返す。
Example
int value = 42;
ket::pointer::NotNull<int> ptr(&value);
// *ptr == 42

Get function

Signature
constexpr T* Get() const noexcept
Brief保持中のraw pointer取得。
Return / retval
  • ptr: 構築時に渡したnon-null raw pointer。
Pre構築でnullを拒否済み。
Postobjectと参照先の状態変更なし。
Example
int value = 42;
ket::pointer::NotNull<int> ptr(&value);
// ptr.Get() == &value

LockWeak function

Signature
template <typename T> std::shared_ptr<T> LockWeak(const std::weak_ptr<T>& weak) noexcept
Briefweak pointerのlock結果取得。
Parameters
  • in weak: lock対象のweak pointer。
Return / retval
  • value: `weak.lock()`で得たshared pointer。
  • empty: `weak`がexpiredのときの空shared pointer。
Pre`weak`は有効なstd::weak_ptr object。
Post`weak` objectと参照先objectの状態変更なし。成功時は返却shared pointerが共有所有権を持つ。
Example
const auto locked = ket::pointer::LockWeak(weak);
// expiredなら空std::shared_ptr
