ket::pointer

所有しないraw pointerのnull不許可、weak pointer lock、address取得を扱う。

Use This When

  • raw pointerがnullでない前提を型名で明示したい時。
  • weak pointerのlock結果やoverloadされたoperator&回避を短く扱いたい時。

Do Not Use This When

  • 所有権を表すsmart pointer wrapperが必要な時。
  • lifetimeやborrow checkingをframeworkとして管理したい時。

API Details

NotNull type

template <typename T> class NotNull

nullを保持しないraw pointer wrapper。

Notes

  • 所有権は持たず、参照先のlifetimeは呼び出し側の責任。
  • `void`は対象外。class instantiation時のstatic_assertで拒否。 参照先型があるraw pointerに限定。

NotNull function

constexpr explicit NotNull(T* ptr)

non-null raw pointerからの構築。

Parameters

  • in ptr: 保持対象のraw pointer。

Contract

prepost
`ptr`はraw pointer。`nullptr`はstd::invalid_argumentとして扱う。構築成功時、`Get()`は`ptr`と同じpointer値を返す。

Example

int value = 42;
ket::pointer::NotNull<int> ptr(&value);
// *ptr == 42

Get function

constexpr T* Get() const noexcept

保持中のraw pointer取得。

Return / retval

  • ptr: 構築時に渡したnon-null raw pointer。

Contract

prepost
構築でnullを拒否済み。objectと参照先の状態変更なし。

Example

int value = 42;
ket::pointer::NotNull<int> ptr(&value);
// ptr.Get() == &value

LockWeak function

template <typename T> std::shared_ptr<T> LockWeak(const std::weak_ptr<T>& weak) noexcept

weak pointerのlock結果取得。

Parameters

  • in weak: lock対象のweak pointer。

Return / retval

  • value: `weak.lock()`で得たshared pointer。
  • empty: `weak`がexpiredのときの空shared pointer。

Contract

prepost
`weak`は有効なstd::weak_ptr object。`weak` objectと参照先objectの状態変更なし。成功時は返却shared pointerが共有所有権を持つ。

Example

const auto locked = ket::pointer::LockWeak(weak);
// expiredなら空std::shared_ptr

AddressOf function

template <typename T> T* AddressOf(T& value) noexcept

overloadされたoperator&を回避した実address取得。

Parameters

  • in value: address取得対象のlvalue。

Return / retval

  • ptr: `value`の実address。

Contract

prepost
`value`は有効なlvalue。`value`と外部状態の変更なし。

Example

const auto ptr = ket::pointer::AddressOf(value);
// operator& overloadを呼ばない
