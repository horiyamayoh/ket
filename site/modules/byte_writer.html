ket::byte_writer

固定bufferへ成功時だけoffsetとbufferを更新してbyte列を書く。

Namespace公開API：ket::byte_writer 内部実装：.cpp の無名 namespace

Use This When

  • 既存固定長bufferへ安全にfieldを書きたい時。
  • 容量不足時にoffsetとbufferを不変にしたい時。

Do Not Use This When

  • 可変長vectorを構築したい時。
  • streamやfileへ直接書きたい時。

Views

バイナリ・byte列固定bufferから安全に読む・書く 小さい電文payloadを組み立てる

API Details

Writer type

class Writer

fixed bufferへ逐次書き込むnon-owning writer。 `data == nullptr && size == 0`は有効な空writer。`data == nullptr && size > 0`は invalid writerとして扱い、書き込みとskipは失敗。

Writer function

Writer(std::uint8_t* data, std::size_t size) noexcept

書き込み先bufferを参照するwriter生成。

Parameters

  • in,out data: 書き込み先buffer先頭。`size == 0`のときだけnullptr可。
  • in size: 書き込み先bufferのbyte数。

Contract

prepost
`data`は`size`バイト以上書き込み可能な配列を指す。`nullptr`と非0 sizeの組はinvalid writerとして構築。offsetは0。buffer内容の変更なし。

Example

std::uint8_t buffer[] = {0x00U, 0x00U};
ket::byte_writer::Writer writer(buffer, 2U);
// writer.Size() == 2
// writer.Offset() == 0

Writer function

Writer(const Writer& other) noexcept = default

writer状態のcopy constructor。

Parameters

  • in other: copy元writer。

Contract

prepost
`other`は有効またはinvalidなWriter object。non-owning pointer、size、offsetをそのまま複製。buffer内容の変更なし。

Example

std::uint8_t buffer[] = {0x00U};

Skip function

bool Skip(std::size_t size) noexcept

指定byte数だけoffsetを進める。

Parameters

  • in size: skipするbyte数。

Return / retval

  • true: 指定byte数のskipに成功。
  • false: invalid writer、または残りbyte数不足。

Contract

prepost
なし。size不足は失敗値として扱う。成功時だけoffsetを`size`だけ進める。buffer内容の変更なし。

Example

std::uint8_t buffer[] = {0x00U, 0x00U};
ket::byte_writer::Writer writer(buffer, 2U);
const auto ok = writer.Skip(1U);
// ok == true
// writer.Offset() == 1

WriteU8 function

bool WriteU8(std::uint8_t value) noexcept

1byte値の書き込み。

Parameters

  • in value: 書き込む値。

Return / retval

  • true: 書き込みに成功。
  • false: invalid writer、または残りbyte数不足。

Contract

prepost
なし。size不足は失敗値として扱う。成功時だけoffsetを1進め、対象byteを更新。失敗時はoffsetとbuffer内容を保持。

Example

std::uint8_t buffer[] = {0x00U};
ket::byte_writer::Writer writer(buffer, 1U);
const auto ok = writer.WriteU8(0x12U);
// ok == true, buffer[0] == 0x12U, writer.Offset() == 1

WriteBytes function

bool WriteBytes(const std::uint8_t* data, std::size_t size) noexcept

byte列の書き込み。

Parameters

  • in data: 書き込むbyte列先頭。`size == 0`のときだけnullptr可。
  • in size: 書き込むbyte数。

Return / retval

  • true: 書き込みに成功。valid writerの`size == 0`はno-op成功。
  • false: null source、invalid writer、または残りbyte数不足。

Contract

prepost
`data`は`size`バイト以上読み取り可能な配列を指す。`nullptr`と非0 sizeの組は失敗値として扱う。成功時だけoffsetを`size`だけ進め、対象範囲を更新。失敗時はoffsetとbuffer内容を保持。

Example

std::uint8_t buffer[] = {0x00U, 0x00U};
const std::uint8_t source[] = {0x10U, 0x20U};
ket::byte_writer::Writer writer(buffer, 2U);
const auto ok = writer.WriteBytes(source, 2U);
// ok == true, buffer[0] == 0x10U, writer.Offset() == 2
