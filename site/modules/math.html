Do Not Use This When

API Details

Lerp function

template <typename T> constexpr T Lerp(T a, T b, T t) noexcept

2値間の線形補間。

Parameters

  • in a: `t == 0`の端点値。
  • in b: `t == 1`の端点値。
  • in t: 補間係数。0から1の範囲外では外挿。

Return / retval

  • value: `t == 0`では`a`、`t == 1`では`b`、それ以外は`a + (b - a) * t`で求めた値。

Contract

prepost
`T`はfloating-point型。引数と外部状態の変更なし。端点以外の丸め、overflow、NaN伝播は通常の浮動小数点演算に従う。

Example

const auto value = ket::math::Lerp(10.0, 20.0, 0.25);
// value == 12.5

NearlyEqual function

template <typename T> constexpr bool NearlyEqual(T a, T b, T epsilon) noexcept

正の絶対許容差による浮動小数点近似比較。

Parameters

  • in a: 比較対象の値。
  • in b: 比較対象の値。
  • in epsilon: 有限で正の絶対許容差。

Return / retval

  • true: `epsilon`が有限の正値で、NaN入力がなく、`a == b`または有限値同士の絶対差が`epsilon`以下。
  • false: `epsilon <= 0`、NaN/Infのepsilon、NaN入力、非等価なInf入力、または絶対差が`epsilon`を超える値。

Contract

prepost
`T`はfloating-point型。引数と外部状態の変更なし。同符号Inf同士は通常の等価比較結果を先に使う。

Example

const auto close = ket::math::NearlyEqual(1.0, 1.001, 0.01);
// close == true

TryBytesFromKiB function

inline bool TryBytesFromKiB(std::uint64_t kib, std::uint64_t& out) noexcept

KiB単位の整数値をbyte数へ変換。

Parameters

  • in kib: KiB単位の値。1 KiBは1024 bytes。
  • out out: 変換後のbyte数。成功時だけ更新。

Return / retval

  • true: `kib * 1024`が`std::uint64_t`へ収まった。
  • false: byte数への変換でoverflowする値。

Contract

prepost
なし。overflowは失敗値として扱う。成功時だけ`out`を更新。失敗時は`out`と外部状態の変更なし。

Example

std::uint64_t bytes = 0;
const auto ok = ket::math::TryBytesFromKiB(4, bytes);
// ok == true, bytes == 4096
