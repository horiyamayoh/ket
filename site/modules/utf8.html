Use This When

  • 入力byte列がUTF-8として妥当か確認したい時。
  • code point数を数えたい時。

Do Not Use This When

  • Unicode正規化、grapheme cluster、case foldingが必要な時。
  • 文字列の所有や変換を行いたい時。

API Details

ValidationResult type

struct ValidationResult

UTF-8検査結果。

`valid`がtrueの場合、入力byte列は妥当なUTF-8で、`error_offset`は0。 `valid`がfalseの場合、`error_offset`は存在するbyteがUTF-8構文または範囲制約に 最初に反した位置。妥当なprefixのまま入力末尾へ達したtruncated sequenceでは、そのsequenceの先頭位置。

IsAscii function

bool IsAscii(std::string_view text) noexcept

ASCII限定byte列の判定。

Parameters

  • in text: 判定対象のbyte列。

Return / retval

  • true: `text`の全byteが`0x00..0x7F`。
  • false: `text`が`0x80`以上のbyteを含む。

Contract

prepost
なし。任意のbyte列を判定対象として扱う。引数と外部状態の変更なし。

Notes

  • 空文字列はASCII限定byte列として扱う。

Example

const auto ascii = ket::utf8::IsAscii("abc");
// ascii == true

Validate function

ValidationResult Validate(std::string_view text) noexcept

UTF-8 byte列の妥当性検査。

Parameters

  • in text: 検査対象のbyte列。

Return / retval

  • result: UTF-8検査結果。妥当な場合は`valid == true`。

Contract

prepost
なし。任意のbyte列を検査対象として扱う。引数と外部状態の変更なし。

Notes

  • 存在するbyteがUTF-8構文または範囲制約に反した場合、そのbyte位置を`error_offset`として返す。
  • 妥当なprefixのまま入力末尾へ達したtruncated sequenceは、そのsequenceの先頭byteを返す。
  • overlong、surrogate、範囲外code point、単独continuation byteはinvalid。

Example

const auto result = ket::utf8::Validate("\xF0\x9F\x98\x80");
// result.valid == true, result.error_offset == 0

IsValid function

bool IsValid(std::string_view text) noexcept

UTF-8 byte列の妥当性判定。

Parameters

  • in text: 判定対象のbyte列。

Return / retval

  • true: `text`が妥当なUTF-8。
  • false: `text`が不正なUTF-8。

Contract

prepost
なし。任意のbyte列を判定対象として扱う。引数と外部状態の変更なし。

Notes

  • `Validate(text).valid`と同義。

Example

const auto valid = ket::utf8::IsValid("\xE3\x81\x82");
// valid == true

CountCodePoints function

std::optional<std::size_t> CountCodePoints(std::string_view text) noexcept

UTF-8 byte列のcode point数取得。

Parameters

  • in text: 数える対象のbyte列。

Return / retval

  • value: UTF-8 code point数。
  • std::nullopt: `text`が不正なUTF-8。

Contract

prepost
なし。任意のbyte列を入力として扱い、不正UTF-8は失敗値として返す。引数と外部状態の変更なし。

Notes

  • grapheme cluster数ではなくUnicode code point数。

Example

const auto count = ket::utf8::CountCodePoints("a\xE3\x81\x82");
// count == std::optional<std::size_t>(2)
