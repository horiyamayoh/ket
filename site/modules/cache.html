ket::cache

手動storage上で値を遅延生成し、再利用と明示破棄を局所化する。

Use This When

  • 必要になった時だけ値を生成し、同じscope内で再利用したい時。
  • heap allocationなしで保持値のaddress stabilityを保ちたい時。

Do Not Use This When

  • thread-safe cacheやglobal registryが必要な時。
  • key付きmemoization frameworkを作りたい時。

API Details

Lazy type

Signature
template <typename T> class Lazy
Brief手動storage上に値を一度だけ遅延生成する非thread-safeなlazy value。
Notes
  • `Lazy`自体はcopy/move禁止。保持値のaddress stabilityを`Reset`まで維持。
  • 再入、同時アクセス、thread-safe cacheは対象外。

Lazy function

Signature
Lazy() noexcept
Brief空状態のlazy value構築。
Pre`T`は`alignas(T)`付きbyte storageで格納可能なobject type。
Post保持値なし。heap allocationなし。
Example
ket::cache::Lazy<int> value;
// value.HasValue() == false

~Lazy function

Signature
~Lazy() noexcept
Brief保持値がある場合の破棄。
Pre`T`のdestructorは例外を投げず、同じ`Lazy`へ再入しない。
Post保持値があった場合はそのlifetime終了。destructor例外は`std::terminate`。
Example
{
    ket::cache::Lazy<int> value;
    value.GetOrCreate([] { return 42; });
}
// held int lifetime has ended

HasValue function

Signature
bool HasValue() const noexcept
Brief保持値の有無判定。
Return / retval
  • true: 保持値あり。
  • false: 保持値なし。
Preなし。
Post`Lazy`と保持値の状態変更なし。
Example
ket::cache::Lazy<int> value;
const auto empty = value.HasValue();
value.GetOrCreate([] { return 42; });
const auto created = value.HasValue();
// empty == false && created == true

Reset function

Signature
void Reset() noexcept
Brief保持値の明示的な破棄。
Return / retval
  • void: 戻り値なし。
Pre`T`のdestructorは例外を投げず、同じ`Lazy`へ再入しない。
Post保持値があった場合はemptyへ戻る。保持値なしの場合は状態変更なし。
Notes
  • 破棄中の例外または再入検出時は`std::terminate`。
Example
ket::cache::Lazy<int> value;
value.GetOrCreate([] { return 42; });
value.Reset();
// value.HasValue() == false

GetOrCreate function

Signature
template <typename Factory> T& GetOrCreate(Factory&& factory)
Brief保持値の取得または遅延生成。
Parameters
  • in factory: 必要時だけ呼び出すfactory object。
Return / retval
  • value: 保持値への参照。
Pre同じ`Lazy`への`GetOrCreate`または`Reset`をfactory実行中に再入呼び出ししない。
Post成功時は保持値あり。factoryまたは`T`構築の例外時はemptyのまま。
Notes
  • 値がある場合、factoryは呼び出しなし。factory objectは`Lazy`内に保持しない。
  • factory実行中の再入検出時は`std::terminate`。
  • 追加heap allocationなし。
Example
ket::cache::Lazy<int> value;
int& first = value.GetOrCreate([] { return 42; });
int& second = value.GetOrCreate([] { return 7; });
// &first == &second && second == 42
